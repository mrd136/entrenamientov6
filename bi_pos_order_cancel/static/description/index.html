POS-Point of Sales Order Cancel, Reverse, Revert and Reset Odoo Apps

POS-Point of Sales Order Cancel Odoo apps is used for cancel done/validated and completed POS order and set it to draft stage. Its very obvious that people or users sometimes make mistakes when working with system, When you validated the POS-Point of sale order from POS screen along with its invoice and picking and made mistakes on quantity or anything else and validate POS order then in by default odoo there is no revert option. POS Order Cancel and Reverse Odoo module helps users to cancel pos order on one click. It cancel POS order along with its picking order as well as invoice.

Features

Cancel and Reset to Draft POS Order on Single Click

Helps to cancel POS order and also reset to draft POS order on single click.

Cancel and Reset to Draft Picking and Invoice of POS Order

Helps to cancel and reset to draft picking order and paid invoice of POS order on single click.

Mass Cancel POS Orders

User can cancel selected POS orders with using mass cancel orders option.

Mass Cancel and Reset to Draft POS Orders

User can cancel and reset to draft selected POS orders with using mass cancel and reset to draft option.
